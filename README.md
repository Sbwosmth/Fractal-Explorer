# 🧠 Опис проєкту

Fractal Explorer — це спеціалізований застосунок для дослідження та візуалізації фрактальних структур. Проєкт розроблений в межах дипломної роботи «Використання концепції фракталів у комп’ютерній графіці».

Програма забезпечує інтерактивну візуалізацію різних типів фракталів, їх дослідження в режимі реального часу, обчислення фрактальної розмірності методом Box-Counting, створення анімацій та аналіз продуктивності обчислень. Застосунок підтримує як ітераційні фрактали (Мандельброта, Жюліа, Ньютона, Фенікс, Burning Ship, Tricorn), так і геометричні та IFS-фрактали (дерево Піфагора, сніжинка Коха, трикутник Серпінського, папороть Барнслі).

Особливістю проєкту є реалізація гібридного режиму обчислень, який використовує GPU-прискорення за допомогою бібліотеки CuPy при наявності сумісної відеокарти NVIDIA та автоматично переходить до виконання обчислень на CPU із застосуванням Numba у випадку відсутності підтримки CUDA.
